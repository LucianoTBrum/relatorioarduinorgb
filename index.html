1. Introdução

Este relatório descreve o projeto de controle de um LED RGB utilizando um potenciômetro em um Arduino, com o objetivo de desenvolver habilidades em eletrônica e programação.

2. Objetivos

O principal objetivo é fazer o controle das cores do LED com a utilização do potenciômetro em um microcontrolador Arduino.

3. Materiais Utilizados

  • Arduino
  • 3 Potenciômetros
  • 3 Resistores (220Ω)
  • Protoboard
  • Fios de conexão

5. Código Fonte


const int potPinR = A0;
const int potPinG = A1;
const int potPinB = A2;

const int ledPinR = 9;
const int ledPinG = 10;
const int ledPinB = 11;

void setup() {
  pinMode(ledPinR, OUTPUT);
  pinMode(ledPinG, OUTPUT);
  pinMode(ledPinB, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  int potValueR = analogRead(potPinR);
  int potValueG = analogRead(potPinG);
  int potValueB = analogRead(potPinB);

  int ledValueR = map(potValueR, 0, 1023, 0, 255);
  int ledValueG = map(potValueG, 0, 1023, 0, 255);
  int ledValueB = map(potValueB, 0, 1023, 0, 255);

  Serial.print("(");
  Serial.print(ledValueR);
  Serial.print(",");
  Serial.print(ledValueG);
  Serial.print(",");
  Serial.print(ledValueB);
  Serial.println(")");
  
  analogWrite(ledPinR, ledValueR);
  analogWrite(ledPinG, ledValueG);
  analogWrite(ledPinB, ledValueB);

  delay(10);
}

6. Procedimentos

Os potenciômetros e o LED RGB foram conectados aos pinos digitais e analógicos do Arduino conforme o diagrama, e o código foi carregado na placa.

7. Resultados

O LED RGB acendeu conforme o esperado, alternando as cores conforme os valores fossem alterados com a utilização dos 3 potenciômetros.

8. Discussão

O funcionamento do circuito aconteceu de forma satisfatória e conforme esperado, alternando suas cores e intensidade.

9. Conclusão

O projeto foi bem-sucedido, permitindo a prática dos conceitos de eletrônica e programação.
