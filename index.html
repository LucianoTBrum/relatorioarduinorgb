1. Introdução

Este Relatório descreve o projeto de um controle do led RGB com a utilização de um potenciometro em um arduino, com o objetivo de desenvolver habilidades em eletronica e programação.

2. Objetivos

O principal objetivo é fazer o controle das cores do led com a utilização do potenciometro em um microcontrolador Arduino.

3. Materiais utilizados

  • Arduino
  • 3 Potenciometros
  • 3 Resistores (220Ω)
  • Protoboard
  • Fios de conexão

5. Código Fonte

            const int potPinR = A0;
            const int potPinG = A1;
            const int potPinB = A2;
            
            const int ledPinR = 9;
            const int ledPinG = 10;
            const int ledPinB = 11;
            
            void setup() {
              pinMode(ledPinR, OUTPUT);
              pinMode(ledPinG, OUTPUT);
              pinMode(ledPinB, OUTPUT);
              Serial.begin(9600);
            }
            
            void loop() {
              int potValueR = analogRead(potPinR);
              int potValueG = analogRead(potPinG);
              int potValueB = analogRead(potPinB);
            
              int ledValueR = map(potValueR, 0, 1023, 0, 255);
              int ledValueG = map(potValueG, 0, 1023, 0, 255);
              int ledValueB = map(potValueB, 0, 1023, 0, 255);
            
              Serial.print("(");
              Serial.print(ledValueR);
                Serial.print(",");
              Serial.print(ledValueG);
                Serial.print(",");
              Serial.print(ledValueB);
              Serial.println(")");
              
              analogWrite(ledPinR, ledValueR);
              analogWrite(ledPinG, ledValueG);
              analogWrite(ledPinB, ledValueB);
            
              delay(10);
            }

6. Procedimentos

Os potenciometros e o led RGB foram conectados aos pinos digitais e analogicos do Arduino conforme o diagrama, e o código foi carregado na placa.

7. Resultados

O led rgb acendeu conforme o esperado, alternando as cores conforme os valores fossem alterados com a utilização dos 3 potenciometros.

8. Discussão

O funcionamento do circuito aconteceu de forma satisfatoria e conforme esperado, alternando suas cores e intensidade.

9. Conclusão

O projeto foi bem sucedido, permitindo a pratica dos conceitos de eletronica e programação.
